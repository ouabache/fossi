CDE LIBRARY:

The Common Design Environment (CDE) is a library of verilog IP modules for use in fpga and asic designs. One problem that we face is that not all rtl code is synthesisable into all target processes. The CDE project seeks to identify this problem code and provide documented and functioning models for each case. A CDE module will isolate the problem code inside a single module that can be easily replaced when the design is targeted to a process that requires substitution of custom hard macros. This can be done without touching the users rtl code so that a single code base can support both fpga and asic targets without modification.



CDE is part of the SOCEN design environment and uses IP-Xact module descriptors. Documention is autogenerated and uses the gEDA tool set.
